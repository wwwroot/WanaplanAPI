registerEventCb

Syntax

registerEventCb

(
  • id
  • priority
  • evName
  • mode
  • objType
  • callback
  • data
)

Summary

Register a callback for a specific event.

Parameters:

  • id String

    An unique identifier for this event (ex: "coreComponentWallClick").

  • priority Number

    The priority for this event (define callback call order).

  • evName String

    The event name associated to callback.

  • mode Number

    Define the mode where the event is active (ex: Core.MODE_NORMAL|Core.MODE_DRAG).

  • objType Object

    The type of object which must be targeted, or null.

  • callback Function

    The function to call.

  • data Object

    (Optional) Data passed to the callback.

requestDynamicDraw

Syntax

requestDynamicDraw

()

Summary

Request a redraw of the top "dynamic" canvas on the next frame The top canvas is used to draw things that may need a refresh when the mouse moves Typically, it is the symbols displayed when the mouse hovers something

requestStaticDraw

Syntax

requestStaticDraw

()

Summary

Request a redraw of the bottom "static" canvas on the next frame The bottom canvas is used to draw things that don't usually need a redraw when the mouse moves, like the walls, the rooms, the grid etc... NB : The top "dynamic" canvas will also be redrawn
