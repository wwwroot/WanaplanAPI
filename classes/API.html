Syntax

addScripts

(
  • scripts
  • options
  • successCallback
  • errorCallback
  • interval
)

Summary

Load a collection of scripts. Here are two methods to load scripts.

  1. A simple array with urls [ "script.js", "script2.js" ]
  2. An array of dictionary [ { MyObject: "myObjectScript.js" } ]. In this case the function will test if the script is already loaded by using the key of the dictionary as object name.

Parameters:

  • scripts Array

    An array that contains all url to load or an array of.

  • options Object

    (Optional) A dictionary of parameters (ex. { "async": "true" }).

  • successCallback Function

    (Optional) A function called when the script is ready.

  • errorCallback Function

    (Optional) A function called if the process fail.

  • interval Number

    (Optional) The value of check interval.

getPickingInfo

(
  • mousePos
)
wnp.BaseStructure

Summary

Get the picking informations with the mouse position. The picking information contain :

- bu, bv : informations on the uv of the picked mesh
- distance : the distance from the camera to the picked point
- faceId : The face id of the mesh
- hit : hit or not a mesh
- pickedMesh : the mesh collided
- pickedPoint : The point on the mesh collided
- pickedSubMeshIndex : The index of the submesh

Parameters:

  • mousePos BABYLON.Vector2

    The mouse position on the screen.

Returns:

wnp.BaseStructure:

The collided informations.

Summary

Request a redraw of the top "dynamic" canvas on the next frame The top canvas is used to draw things that may need a refresh when the mouse moves Typically, it is the symbols displayed when the mouse hovers something

requestStaticDraw

Syntax

requestStaticDraw

()

Summary

Request a redraw of the bottom "static" canvas on the next frame The bottom canvas is used to draw things that don't usually need a redraw when the mouse moves, like the walls, the rooms, the grid etc... NB : The top "dynamic" canvas will also be redrawn
