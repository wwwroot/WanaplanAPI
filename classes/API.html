Summary

    The target object, can be anything that is concerned by the action.

compareVersion

Syntax

compareVersion

(
  • versionA
  • versionB
)
Number

Summary

Compares two stringified versions.

Parameters:

  • versionA String

    The first version.

  • versionB String

    The second version.

Returns:

Number:

Returns -1 if versionA is older than versionB. Returns 0 if versionA is the same as versionB. Returns 1 if versionA is newer than versionB.

setMode

Syntax

setMode

(
  • name
  • [_context]
)

Summary

Sets the mode for a given context

Parameters:

  • name String

    Must be one of the following :

                - for 2D : API.MODE_2D_NORMAL, API.MODE_2D_DRAG, API.MODE_2D_DRAW, API.MODE_2D_CONTEXTMENU, API.MODE_2D_SUBSLOPE
            - for 3D : API.MODE_3D_NORMAL, API.MODE_3D_PAINT
  • [_context] Number optional

    The context, 2D or 3D. Default is the currently selected context.
