DragComponent3D

Extends BaseComponent3D
Module: Component
Parent Module: Wanaplan

Summary

This component is responsible for handling the object moving by drag an drop. Listen the drag envent, resolve it by detecting the object under the cursor. When a draggable object is draggend, resolve the new position computation using the projection of the cursor on the scene. Then apply the move using the transformComponent3D

  • floorY Number

    elevation of the floor

Events

wnp.engine3D.drag-start

Syntax

wnp.engine3D.drag-start

Summary

[in] Fired by the engine when the user start dragging its cursor, /!\ it's just about the cursor, it's not necessary an object which dragged when capted, test is an object is below the cursor, if so start moving it

wnp.engine3D.drag-start

Syntax

wnp.engine3D.drag-start

Summary

[in] Fired by the engine when the user is dragging its cursor, /!\ it's just about the cursor, it's not necessary an object which dragged when capted, if an object was below the cursor when the user start dragging, determine the new position of the object and apply it with transformComponent3D

wnp.engine3D.drag-start

Syntax

wnp.engine3D.drag-start

Summary

[in] Fired by the engine when the user stop dragging its cursor when capted, if an object was dragged, stop to drag it, notify that the transformation is over
