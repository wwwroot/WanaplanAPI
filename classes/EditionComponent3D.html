EditionComponent3D

Module: Component
Parent Module: Wanaplan

Summary

Component managing the user-guided edition of the 3D scene. Manages all the interactions involving the modifications of the scene's geometry. Necessary when the scene can be edited by the user.

For instance, in viewer mode, this component is inactive.

  • locker
  • flags
)

Summary

lock some actions. Register a key so that the action can only be re-able with the same key. actions to lock must be passed as an addition of their corresponding flag support multi-lock :

A lock the action ------------------------------------------------------------------- A unlock the action B lock the same action ----------------- B unlock the action

<-------------------------------action is locked ------------------------------------------------------->

--------time--------->

Parameters:

  • locker Object

    can be anything, is register as key for the lock, the action will be unlock only with the same key

  • flags Number

    [ optional ] actions to lock, CAMERALOCK = 1; DRAGLOCK = 2; KEYBOARDLOCK = 4; CAMERATRANSLATIONLOCK = 8;
