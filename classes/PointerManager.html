PointerManager

Module: Input
Parent Module: Wanaplan

Summary

The pointer manager is an abstraction of MouseEvent, TouchEvent and PointerEvent and is responsible to manage all type of 'pointer' input in Wanaplan.

Constructor

PointerManager

Syntax

PointerManager

(
  • core
  • callback
  • domElement
  • param
)

Summary

Parameters:

  • core Core

    The main engine.

  • callback Function

    Function called when an event is intercepted (can be null).

  • domElement HTMLElement

    The DOM element to use for listening events.

  • param Object

    (optional) Additional parameters { offsets: BABYLON.Vector3(0,0), width: 0, height: 0 }.
