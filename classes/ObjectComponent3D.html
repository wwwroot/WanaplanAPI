buildObject

Syntax

buildObject

(
  • objectStructure
  • params
  • [floorStructure]
  • [callback]
  • [silent=false]
)

Summary

Builds the object, given the object structure and the object params. As this may require to load scripts, the function is asynchronous. Appends the object to the floor when it is ready, throws the wnp.engine3D.object.create event when it's done.

Parameters:

  • objectStructure ObjectStructure

    The object structure to be instanciated.

  • params Object

    Params of the object to be instanciated such as defined in the specific programmable.

    • [materials] BABYLON.Material optional

      Contains labeled materials. If the materials are already defined in objectStructure.programmableInstance, this will have no effect.

  • [floorStructure] FloorStructure optional

    The floor where the object will be appended, default is the current floor.

  • [callback] Function optional

    The function called when the object is built.

  • [silent=false] Boolean optional

    If true, no event will be thrown at the object creation.

refreshObject

Syntax

refreshObject

(
  • meshObject
  • [callback]
)

Summary

Rebuilds the object, assuming the structure object have changed (not necessarely the case). The object reference will be kept. This function is asynchronous. Throws wnp.engine3D.object.refresh when the object mesh have been rebuilt.

Parameters:

  • meshObject BABYLON.Mesh

    The mesh object, as defined in wanaplan, should declare a reference to the objectStructure ( meshObject.structure ).

  • [callback] Function optional

    The function called when the object is built. Takes the new mesh as its only parameter.
