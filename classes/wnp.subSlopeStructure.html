checkCoherence

()

Checks the coherency of the structure, for instance : 

 - this.lowHeight < this.hiHeight
 - this.lowHeight > 0
 - this.hiHeight < 4000

WARNING : This method automatically corrects incoherency. So it may modify the parameters of the subslope.
