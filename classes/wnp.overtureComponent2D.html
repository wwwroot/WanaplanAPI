Provided by the Wanaplan module.

rotate the object on the scene

Parameters:

  • object BABYLON.Mesh

    the object

  • newRotation BABYLON.Vector3

    the new rotation

  • options Object

    [optional] set of options

    • noMagnetism Boolean

      if true, the magnetism will not be applied

    • noHistory Boolean

      if true, the transformation will not be historized

    • silent Boolean

      if true, the transformation will not trigger the event

    • relative Boolean

      if true, the newPosition will be added to the current rotation

    • old BABYLON.Vector3

      the old rotation, if omited take the current rotation

moveObject

(
  • object
  • newPosition
  • options
)

Provided by the Wanaplan module.

move the object on the scene

Parameters:

  • object BABYLON.Mesh

    the object, can be a group

  • newPosition BABYLON.Vector3

    the new position

  • options Object

    [optional] set of options

    • noMagnetism Boolean | String

      if true, the magnetism will not be applied, if "xz" the xz magnetism will not be applied, same thing for "y"

    • noHistory Boolean

      if true, the transformation will not be historized

    • silent Boolean

      if true, the transformation will not trigger the event

    • relative Boolean

      if true, the newPosition will be added to the current position

    • enableGravity Boolean

      if true, the object bottom have to touch another object along the y axis

    • old BABYLON.Vector3

      the old position, if omited take the current position

Events

wnp.engine3D.object.end-transform

Provided by the Wanaplan module.

The object in params will not longer be transformed a lot ( well at least that the spirit )

Event Payload:

  • object BABYLON.Mesh

    the object

    throw when 

    * endTransformObject is callled

    if the object have been transformed since the last beginTransform, wnp.engine3D.object.translate and wnp.engine3D.object.translate can be thown too

wnp.engine3D.object.rotate

Provided by the Wanaplan module.

The object in params have been rotated

Event Payload:

  • object BABYLON.Mesh

    the object

  • pending Boolean

    true if the object is in a begin - end period

    throw when 

    * moveObject is called, if the magnetism have decided that it should be rotated
* rotateObject is called
* endTransformObject is called, and a rotation have been performed since the last beginTransform

wnp.engine3D.object.start-transform

Provided by the Wanaplan module.

The object in params will be transformed a lot ( well at least that the spirit )

Event Payload:

  • object BABYLON.Mesh

    the object

    throw when 

    * beginTransformObject is callled

wnp.engine3D.object.translate

Provided by the Wanaplan module.

The object in params have been translated

Event Payload:

  • object BABYLON.Mesh

    the object

  • pending Boolean

    * moveObject is callled
