wnp.Configuration Class

Module: Core
Parent Module: Wanaplan

Methods

getConfiguation

() String

Serialize the configuration into a string.

Returns:

String:

Returns a serialized string of the configuration.

loadConfiguration

() Boolean

Load uers's preference from LocalStorage. If no data are available, default values are used.

Returns:

Boolean:

Returns ’true’ if data has been successfully loaded, otherwise it returns false.

saveConfiguration

()

Save user preferences into LocalStorage.

setQuality

(
  • quality
)

Change the graphics quality and save the configuration.

wnp.Constants.GRAPHICS_FAST:

 - Anti-Aliasing, Shadows, Environment Map, Multi-Texturing, Multi-Lighting = false
 - Hardware Scaling = 1.5.

wnp.Constants.GRAPHICS_GOOD:

 - Anti-Aliasing, Multi-Texturing, Multi-Lighting = true, Environment Map, Shadows = false
 - Hardware Scaling = 1.0.

wnp.Constants.GRAPHICS_QUALITY:

 - Anti-Aliasing, Shadows, Environment Map, Multi-Texturing, Multi-Lighting = true
 - Hardware Scaling = 1.0.

Parameters:

  • quality Number

    The desired level of quality.
